## üöÄ How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/KyneLaggui/emabini-quiz.git
   ```

2. Navigate into the project directory:

   ```bash
   cd emabini-quiz
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm start
   ```

5. Open your browser and visit:
   ```
   http://localhost:3000
   ```
